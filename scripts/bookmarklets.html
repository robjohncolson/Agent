AI Studio Bookmarklets

AP Stats Lesson Prep

How to install

  1. Make sure your bookmarks bar is visible (Ctrl+Shift+B in Chrome/Edge).
  2. Drag each colored button below onto your bookmarks bar.
  3. Navigate to AI Studio, upload a lecture video, then click the bookmarklet.
  4. The prompt is auto-filled. Click Send when ready.

Step 1 — Fill the transcription prompt drag to bookmarks bar

📝 Transcribe

Fills the AI Studio prompt box with a timestamped transcription request.

Transcribe this video with timestamps. Format each segment as: **[MM:SS]** <transcribed text> Include all spoken content. Be thorough and accurate.

Step 2 — Fill the slide-description prompt drag to bookmarks bar

🖼 Slides

Fills the prompt with a detailed slide-by-slide visual description request.

Describe each slide or visual change in this video with timestamps. Format as: **[MM:SS]** — **Slide title or topic** <Description of what's shown: text, formulas, graphs, diagrams, examples, key definitions> Be thorough — capture all text on each slide, any formulas, graph labels, and visual details that a student would need to follow along.

Step 3 — Save the model response to a file drag to bookmarks bar

💾 Save Output

Extracts the last model response and downloads it as a text file. You will be prompted for a filename.

Works on any browser — no extensions needed, cannot be blocked by admin.

Bookmarklets run in the page context. They are standard browser functionality, not third-party code.
